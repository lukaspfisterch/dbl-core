# Constitution

## What This Is

dbl-core is a deterministic event substrate for the Deterministic Boundary Layer.

## Governing Principles

1. **Only DECISION events are authoritative.**
2. **Append-only event stream V.**
3. **Contract-first.** Code conforms to the contract, not the other way around.
4. **No domain semantics.** Domain logic belongs in domainrunners.

## What This Is Not

- Not a workflow or policy engine.
- Not an execution orchestrator.
- Not an LLM wrapper.
